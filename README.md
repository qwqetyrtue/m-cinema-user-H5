# m-cinema-user

## Describe `描述`

>  m-cinema 用户手机端webH5页面

### 技术

+ `van2`
+ `vue-2.6.14`
+ `vuex`
+ `vue-router`

### 结构

与`vue-element-admin`项目结构类似

> ├── src
> 	└── api  // 接口访问
> 	└── App.vue // 启动
> 	└── assets // 动态资源
> 	└── components // 全局组件
> 	└── layout // 布局
> 	└── main.js // 启动
> 	└── permission.js // 权限验证
> 	└── router // 路由管理
> 	└──  store // 全局存储
> 	└── utils // 工具类
> 	└──  views // 视图页面
